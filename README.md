# Blinkit---Sales---Analysis
An interactive retail analytics dashboard built to analyze sales performance, customer satisfaction, and inventory distribution across Blinkit outlets. The dashboard focuses on identifying key sales drivers, outlet performance, and product trends to support data-driven decision making.

2. Short Description / Purpose 
The Blinkit Sales Analysis Dashboard is designed to explore sales trends across different outlet types, sizes, and locations. It helps identify top-performing products, customer preferences, and opportunities for improving sales and inventory management.

3. Tech Stack
The dashboard was built using the following tools and technologies:
Power BI Desktop – Data visualization and dashboard creation
Power Query – Data cleaning and transformation
DAX – Calculated measures and KPIs
Data Modeling – Relationships between sales and outlet data
File Format – .pbix for development and .png for preview

4. Data Source
Dataset: Blinkit Grocery Sales Dataset
The dataset contains information about:
• Item Type
• Fat Content
• Outlet Size
• Outlet Location
• Outlet Type
• Sales Amount
• Customer Ratings
• Number of Items
The data is structured to analyze product-level sales and outlet-level performance.

5. Features / Highlights
Business Problem

Retail businesses like Blinkit handle thousands of products across multiple outlets. It becomes difficult to quickly answer questions like:

• Which outlet type generates highest sales?
• Which item categories perform best?
• Does outlet size impact revenue?
• How customer ratings affect sales?
• Which locations perform better?

Without a dashboard, these insights are hard to identify.

Goal of the Dashboard

To create an interactive dashboard that:

• Tracks overall sales performance
• Identifies top-performing item categories
• Compares outlet performance
• Analyzes customer ratings impact
• Provides actionable business insights

Business Impact & Insights

• Identified top-performing outlet types driving revenue
• Found customer preference between Low Fat and Regular items
• Highlighted best-selling product categories
• Compared outlet performance by size and location
• Provided insights for inventory and sales optimization

This dashboard helps retail businesses improve decision-making, optimize inventory, and increase sales performance.
